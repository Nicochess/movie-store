<h1 align="center">Movie Store</h1>

<p align="center">O projeto "Movie Store" é uma pequena loja de filmes para você conhecer tudo sobre cinema!</p>

<p align="center">Todo o layout foi estilizado e idealizado por mim, utilizei o padrão BEM para nomenclatura das classes, Mobile First para melhor aproveitamento do código, fiz o Deploy utilizando GitHub Pages.</p>

```
git clone https://github.com/Nicochess/fullstack-project.git
```

- Para instalar as dependências do projeto, digite em seu terminal
```
npm install
```

- Para subir localmente
```
npm start
```
